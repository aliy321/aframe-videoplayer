# Aframe video player

A webVR video player made quickly with [A-Frame](https://aframe.io).
It allows the user to play, pause, mute, rewind and seek.

#### [Live demo](https://etiennepinchon.github.io/aframe-videoplayer/)

### Install/Run
```
npm install && npm start
```
